2. Andet kapitel

Kopiér kapitel-blokken for flere kapitel. Skift data-chapter, header-klasse (section-beige / section-blue / section-green / section-teal) og body-klasse (ingen / alt-bg / alt-bg-2).

Kun det første kapitel skal have class="chapter active"; resten kun class="chapter".
